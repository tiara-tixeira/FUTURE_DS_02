
# FUTURE_DS_02
Business sales data analysis project using Power BI to uncover revenue trends, product performance, regional insights, and actionable business recommendations.

##Project Overview

This project was completed as part of the Data Science & Analytics track. The objective was to analyse business sales data and transform it into meaningful insights that can support better business decision-making.
An interactive dashboard was created using Microsoft Power BI to analyse sales performance across different years, regions, and product categories.

##Business Objective

The objective of this project is to help answer important business questions such as:
Which products generate the most revenue?
How do sales change over time?
Which categories perform the best?
Which regions contribute the most to business performance?
Where should the business focus to improve growth?
🛠️ Tools Used
Microsoft Power BI — Data visualization, dashboard creation, and interactive analysis
Microsoft Excel — Data cleaning and preparation

##Dashboard Features

The interactive dashboard includes:
Total Sales KPI
Total Profit KPI
Total Orders KPI
Total Quantity KPI
Sales and revenue trend analysis
Category-wise performance analysis
Regional performance analysis
Interactive Year filter
Interactive Region filter
Interactive Category filter
Business insights and recommendations

##Key Insights

The dashboard enables analysis of:
Overall business performance through key performance indicators
Changes in sales and revenue over time
Performance across different product categories
Regional differences in business performance
High-performing and underperforming areas

##Business Recommendations

Based on the analysis, businesses can:
Focus on high-performing products and categories
Identify regions with strong growth potential
Investigate underperforming categories or regions
Monitor sales trends to support future planning
Use data-driven insights to improve business decision-making

##Conclusion

This project demonstrates how raw business sales data can be transformed into interactive visualizations and actionable business insights using Power BI.
